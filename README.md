# Aprenda Huawei Cloud BR

Este repositório contém uma página HTML estática que agrega os principais links
para documentações, materiais de treinamento e certificações da Huawei Cloud,
disponíveis em português e em inglês.

Baseado em [LittleLink](https://github.com/sethcottle/littlelink/), uma
alternativa de código aberto ao Linktree.
